Fringe reduction algorithm for absorption images

When studying ultracold atoms, we commonly take images of the atom cloud. This is an implementation of a noise suppression algorithm for absorption images of ultracold atoms. In a nutshell, a series of background images are taken, containing only the noise. The algorithm then decomposes the noise present in these images into an orthonormal basis (through Schmidt-Decomposition). With this basis, an idealized reference image can be constructred to cancel out the noise in the images with the atoms (the signal). The key reason why this approach works is that there is an inherent structure in the noise.
